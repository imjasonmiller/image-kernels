# Image kernels

A small playground for experimenting with image kernels and other effects


## Install 

To get started, clone the repository:

```sh
git clone https://github.com/imjasonmiller/image-kernels.git
```

To run:

```sh
$ npm i
$ npm start
```

To test the code:

```sh
$ npm run test
```
